
# 📝 MemoFlow

## Wireframe - Versión 1.0
## Comportamiento de las notas

### Estado contraído

- Se muestra únicamente el título.
- Diseño limpio y minimalista.

### Estado expandido

Al hacer clic sobre una nota se despliega:

- Descripción
- Fecha (si existe)
- Hora (si existe)

Acciones disponibles:

- ☐ Marcar como completada
- ✏ Editar
- 🗑 Eliminar

La expansión tendrá una animación suave.

## Interacción responsive

- Toda la tarjeta será clickeable.
- La flecha funcionará únicamente como indicador visual.
- En dispositivos móviles habrá una zona táctil amplia.
- Solo una nota podrá permanecer expandida al mismo tiempo. 

## Responsive

MemoFlow será diseñado siguiendo la filosofía Mobile First.

Las acciones de cada nota se mostrarán en columna para facilitar su uso en dispositivos móviles.

Cada botón tendrá una zona táctil amplia para mejorar la experiencia del usuario.

## Filosofía de la interfaz

MemoFlow solo muestra información que aporta valor.

Si un dato no existe, no se muestra ningún texto de reemplazo.

La interfaz debe mantenerse limpia y enfocada en el contenido.
